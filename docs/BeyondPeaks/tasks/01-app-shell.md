## 交付物

交付一个可运行的 Flutter 应用壳：应用能启动，使用 `go_router` 的 `ShellRoute` 承载底部导航，四个入口 `首页 / 攀爬 / 收藏 / 设置` 可切换，其中首页承载后续 3D 内容，其余三个入口先显示占位页。主题需要接入阶段一的 Stone & Slate 设计规范，底部导航的当前选中态要清晰可见。

## 验收标准

- [ ] App 启动后能进入主界面，底部导航可见。
- [ ] `首页 / 攀爬 / 收藏 / 设置` 四个入口可切换，路由行为稳定。
- [ ] 攀爬、收藏、设置页面先显示明确占位内容，不报错、不空白。
- [ ] 主题样式与 `DESIGN.md` 的阶段一视觉约束一致。
