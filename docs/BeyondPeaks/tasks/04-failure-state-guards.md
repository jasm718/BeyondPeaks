## 交付物

交付阶段一的失败态与数据守卫：`earth.glb` 缺失或加载失败时，首页 3D 区域显示明确失败态但底部导航仍可用；山峰数据为空时显示空状态；非法经纬度数据在开发期直接抛错阻止进入渲染；3D 初始化失败时不进入交互逻辑。

## 验收标准

- [x] `earth.glb` 缺失或加载失败时，首页 3D 区域显示失败态。
- [x] 3D 区域失败后，底部导航仍可切换其他页面。
- [x] 山峰数据为空时，首页显示空状态而不是空白或崩溃。
- [x] 非法经纬度数据会被立即拒绝，不进入渲染流程。
- [x] 3D 初始化失败时，页面不会进入错误交互状态。
